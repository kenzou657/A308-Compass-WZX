```
┌─────────────────────────────────────────────────────────────────┐

│  Timer 溢出  │  UART 接收  │  GPIO 变化  │  ADC 完成            │
└─────────────────────────────────────────────────────────────────┘
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│              ISR 处理 (src/isr/isr_handlers.c)                  │
├─────────────────────────────────────────────────────────────────┤
│  ✓ 设置标志位 (g_timer_flag, g_uart_frame_ready)               │
│  ✓ 更新计数器 (整型运算)                                        │
│  ✓ 队列数据 (UART 缓冲)                                         │

│              主循环检查标志位 (src/main.c)                       │
├─────────────────────────────────────────────────────────────────┤
│  if (g_timer_flag) {                                            │
│    // 处理定时器事件                                            │
│    line_tracker_update();  // 调用应用层                        │
│  }                                                              │
│                                                                 │
│  if (g_uart_frame_ready) {                                      │
│    // 处理 UART 数据                                            │
│    digit_recognizer_process_uart_data(...);                    │
│  }                                                              │
└─────────────────────────────────────────────────────────────────┘
```

## 数据流示例：循迹控制

```
┌──────────────────────────────────────────────────────────────────┐
│ 1. Timer ISR (每 10ms 触发)                                      │
│    └─ 设置 g_timer_flag = 1                                     │
└──────────────────────────────────────────────────────────────────┘
                              ▼
┌──────────────────────────────────────────────────────────────────┐
│ 2. 主循环检查标志                                                │
│    if (g_timer_flag) {                                           │
│      g_timer_flag = 0;                                           │
│      ...                                                         │
│    }                                                             │
└──────────────────────────────────────────────────────────────────┘
                              ▼
┌──────────────────────────────────────────────────────────────────┐
│ 3. 驱动层：读取传感器                                            │
│    int16_t sensor_val = line_sensor_read();                     │
│    // 返回 -100 ~ 100 (偏离度)                                  │
└──────────────────────────────────────────────────────────────────┘
                              ▼
┌──────────────────────────────────────────────────────────────────┐
│ 4. 应用层：PID 计算                                              │
│    float error = sensor_val - target;                           │
│    float motor_cmd = pid_update(&pid, error);                  │
│    // 返回 -100 ~ 100 (电机占空比)                              │
└──────────────────────────────────────────────────────────────────┘
                              ▼
┌──────────────────────────────────────────────────────────────────┐
│ 5. 驱动层：执行控制                                              │
│    motor_set_speed(motor_cmd);                                  │
│    // 更新 PWM 占空比                                            │
└──────────────────────────────────────────────────────────────────┘
```

## 模块接口规范

### 驱动层接口模板

```c
// motor_driver.h
#ifndef MOTOR_DRIVER_H
#define MOTOR_DRIVER_H

#include <stdint.h>

/**
 * 初始化电机驱动
 * 配置 PWM 定时器、GPIO 等
 */
void motor_init(void);

/**
 * 设置电机速度
 * @param speed: -100 ~ 100 (负数为反向)
 */
void motor_set_speed(int16_t speed);

/**
 * 获取当前电机速度
 * @return: -100 ~ 100
 */
int16_t motor_get_speed(void);

/**
 * 停止电机
 */
void motor_stop(void);

#endif
```

### 应用层接口模板

```c
// line_tracker.h
#ifndef LINE_TRACKER_H
#define LINE_TRACKER_H

#include <stdint.h>

/**
 * 初始化循迹控制器
 * 配置 PID 参数等
 */
void line_tracker_init(void);

/**
 * 更新循迹控制
 * 应在主循环中定期调用（如每 10ms）
 */
void line_tracker_update(void);

/**
 * 获取电机控制命令
 * @return: -100 ~ 100 (电机占空比)
 */
int16_t line_tracker_get_motor_cmd(void);

/**
 * 设置循迹目标
 * @param target: 目标偏离度 (-100 ~ 100)
 */
void line_tracker_set_target(int16_t target);

#endif
```

## 关键注意事项

### ⚠️ SysConfig 配置
- 修改 `empty.syscfg` 中的输出路径
- 在 Keil 中重新生成配置
- 不要手动编辑生成的 `ti_msp_dl_config.c/h`

### ⚠️ Include 路径
- 确保 Keil 项目的 Include Paths 包含所有源目录
- 使用相对路径（如 `.\src\drivers`）

### ⚠️ 编译顺序
- 驱动层必须先编译（应用层依赖）
- ISR 处理器必须与驱动层一起编译

### ⚠️ 中断优先级
- 在 SysConfig 中设置，不在代码中设置
- 确保 Timer ISR 优先级高于 UART ISR

### ⚠️ 内存管理
- 32KB SRAM 限制
- 避免大型全局数组
- 使用栈上的局部变量

## 性能优化建议

### 1. 整型缩放替代浮点
```c
// 不推荐
float angle = 45.5;  // 浮点运算慢

// 推荐
int16_t angle = 4550;  // 表示 45.50°，整型运算快 10-100 倍
```

### 2. ISR 最小化
```c
// 不推荐
void TIMER_ISR(void) {
    float result = pid_update(&pid, error);  // 浮点运算
    motor_set_speed(result);                 // 函数调用
}

// 推荐
void TIMER_ISR(void) {
    g_timer_flag = 1;  // 仅设置标志
}
```

### 3. 缓冲区管理
```c
// 使用环形队列缓冲 UART 数据
typedef struct {
    uint8_t buffer[256];
    uint16_t head, tail;
} CircularQueue_t;
```

## 编译验证步骤

1. **清理旧编译**
   ```
   Keil → Project → Clean
   ```

2. **更新 Include Paths**
   ```
   Project → Options for Target → C/C++ → Include Paths
   添加: .\src .\src\config .\src\drivers .\src\app .\src\utils .\src\isr
   ```

3. **添加源文件**
   ```
   Project → Manage Project Items
   添加所有 .c 文件
   ```

4. **编译**
   ```
   Keil → Build Target
   ```

5. **验证输出**
   ```
   检查 keil/Objects/ 中的 .axf 和 .hex 文件
   ```

## 常见问题排查

| 问题 | 原因 | 解决方案 |
|------|------|--------|
| 编译错误：找不到头文件 | Include Path 不正确 | 检查 Keil 项目设置 |
| 链接错误：未定义的引用 | 源文件未添加到项目 | 在 Keil 中添加 .c 文件 |
| SysConfig 配置不生效 | 未重新生成配置 | 在 Keil 中重新运行 SysConfig |
| 烧写后无反应 | 主程序入口错误 | 检查 `src/main.c` 中的 `main()` 函数 |
| 中断不工作 | ISR 优先级冲突 | 在 SysConfig 中检查中断优先级 |
