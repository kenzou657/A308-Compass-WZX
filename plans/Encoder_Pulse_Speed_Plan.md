# 编码器脉冲读取与速度计算方案

## 📋 系统配置总结

### 硬件连接
| 项目 | 电机1 (E1A) | 电机2 (E2A) |
|------|-----------|-----------|
| A相捕获 | TIMG0 (PA12) | TIMA1 (PA15) |
| B相GPIO | PB20 | PB25 |
| 时钟频率 | 10MHz | 20MHz |
| 捕获模式 | 上升沿 | 上升沿 |

### 时钟定时器 (CLOCK)
- **定时器**：TIMA0
- **周期**：10ms
- **时钟频率**：100kHz (BUSCLK/8, 分频系数100)
- **中断**：ZERO_EVENT (计数器溢出)

## 🔧 核心算法

### 1. 脉冲计数逻辑
```
E1A/E2A 中断处理流程：
┌─────────────────────────────────────┐
│ A相上升沿中断触发                     │
├─────────────────────────────────────┤
│ 读取B相电平                          │
├─────────────────────────────────────┤
│ IF B相 = 1:                         │
│    脉冲计数 += 1  (正转)             │
│ ELSE:                               │
│    脉冲计数 -= 1  (反转)             │
└─────────────────────────────────────┘
```

**说明**：
- 使用 `int16_t` 存储计数，支持正反转
- 计数范围：-32768 ~ +32767
- 每10ms清零一次（由CLOCK中断触发）

### 2. 速度计算逻辑
```
CLOCK 中断处理流程（10ms周期）：
┌─────────────────────────────────────┐
│ 保存当前脉冲计数                      │
├─────────────────────────────────────┤
│ 计算原始速度：                        │
│ raw_speed = pulse_count (脉冲/10ms) │
├─────────────────────────────────────┤
│ 一阶低通滤波：                        │
│ speed_filtered = α*raw_speed +      │
│                  (1-α)*speed_prev   │
├─────────────────────────────────────┤
│ 转换为线速度（mm/s）：                │
│ v = (speed_filtered / PPR) *        │
│     wheel_circumference * 100       │
│   (100 = 1000ms/10ms)               │
├─────────────────────────────────────┤
│ 清零脉冲计数器                        │
└─────────────────────────────────────┘
```

**参数说明**：
- `PPR`：编码器分辨率（脉冲/转）
- `wheel_circumference`：轮子周长（mm）
- `α`：滤波系数（0.0~1.0，推荐0.3~0.5）

### 3. 速度转换公式
```
线速度 (mm/s) = (脉冲数 / PPR) × 轮周长(mm) × 100
              = (脉冲数 × 轮周长 × 100) / PPR

示例（PPR=20, 轮周长=100mm）：
- 10脉冲/10ms → (10 × 100 × 100) / 20 = 5000 mm/s = 5 m/s
- 1脉冲/10ms  → (1 × 100 × 100) / 20 = 500 mm/s = 0.5 m/s
```

## 📝 配置参数（config.h）

需要在 `config.h` 中添加以下宏定义：

```c
/* ============================================================================
 * 编码器配置
 * ============================================================================ */

/** 编码器分辨率 (脉冲/转) */
#define ENCODER_PPR                 20

/** 轮子周长 (mm) */
#define WHEEL_CIRCUMFERENCE_MM      100

/** 编码器采样周期 (ms) - 由CLOCK定时器决定 */
#define ENCODER_SAMPLE_PERIOD_MS    10

/** 一阶低通滤波系数 (0.0~1.0) */
#define SPEED_FILTER_ALPHA          0.4f

/** 速度单位转换系数 */
#define SPEED_SCALE_FACTOR          ((WHEEL_CIRCUMFERENCE_MM * 100) / ENCODER_PPR)
```

## 🔌 集成步骤

### 1. 初始化流程
```c
// 在 main() 中调用
SYSCFG_DL_init();           // SysConfig初始化（已包含E1A、E2A、CLOCK）
MotorInit();                // 电机驱动初始化
EncoderInit();              // 编码器驱动初始化
DL_TimerA_startCounter(CLOCK_INST);  // 启动CLOCK定时器
```

### 2. 中断处理
```c
// isr_encoder.c 中实现
void TIMG0_IRQHandler(void)     // E1A中断
void TIMA1_IRQHandler(void)     // E2A中断
void TIMA0_IRQHandler(void)     // CLOCK中断
```

### 3. 主循环使用
```c
// 在主循环中读取速度
int16_t speed_motor_a = EncoderGetSpeed(ENCODER_A);  // mm/s
int16_t speed_motor_b = EncoderGetSpeed(ENCODER_B);  // mm/s

// 用于PID控制
pid_feedback_a = speed_motor_a;
pid_feedback_b = speed_motor_b;
```

## 📌 注意事项

1. **中断优先级**：
   - E1A/E2A ISR：高优先级（快速响应）
   - CLOCK ISR：中优先级（允许被E1A/E2A打断）

2. **数据一致性**：
   - 在主循环中读取速度时，可能被CLOCK中断打断
   - 建议使用原子操作或临界区保护

3. **整数溢出**：
   - 脉冲计数使用 `int16_t`，范围 -32768~+32767
   - 10ms内最多计数32脉冲（对应~1.6m/s），不会溢出

4. **滤波参数调整**：
   - α=0.3：滤波强度大，响应慢
   - α=0.5：平衡滤波和响应
   - α=0.7：滤波强度小，响应快
