# AGENTS.md

This file provides guidance to agents when working with code in this repository.

## Code Mode - Project-Specific Rules

### SysConfig Integration (Critical)
- **Never manually edit** `ti_msp_dl_config.c/h` - these are auto-generated by SysConfig
- **Always regenerate** after modifying `empty.syscfg` in Keil SysConfig editor
- **BeforeMake hook** in Keil automatically runs `syscfg.bat` - do not bypass this

### DriverLib Patterns
- **Mandatory**: Use DriverLib functions only (e.g., `DL_TimerG_setCaptureCompareValue()`)
- **Forbidden**: Direct register manipulation (e.g., `TIMER->CCR = value`)
- **Exception**: Only if performance profiling proves DriverLib insufficient

### Module Architecture
- **Driver Layer**: Hardware abstraction (GPIO, UART, PWM control)
- **App Layer**: Business logic (PID, line tracking, digit recognition)
- **Decoupling**: Use function pointers or callbacks, never call App from Driver ISR
- **PID Logic**: Must NOT be in Timer ISR - use flag-based state machine instead

### Interrupt Handling
- **Priority Configuration**: Set in SysConfig, not in code
- **ISR Pattern**: Keep ISRs minimal - set flags, defer processing to main loop
- **UART Reception**: Use interrupt or DMA mode (never polling)
- **Frame Validation**: Always verify frame header/footer (e.g., 0x55, 0xAA)

### Naming & Comments
- **Language**: All comments, variable names in Chinese (中文)
- **Prefix Convention**: 
  - `g_` for global variables
  - `s_` for static variables
  - `p_` for pointers
- **Example**: `g_motorPWM`, `s_pidState`, `p_uartBuffer`

### Performance Constraints
- **Floating Point**: Minimize in critical loops - use integer scaling instead
- **Example**: Store angles as `int16_t` (0-36000 for 0-360°) instead of `float`
- **Clock Tree**: 80MHz MCLK configured in SysConfig - verify before optimization

### File Organization
- `empty.c` - main entry point, calls `SYSCFG_DL_init()`
- `ti_msp_dl_config.c/h` - auto-generated, do not edit
- `keil/` - Keil project files, build outputs in `keil/Objects/`
- `doc/` - contest requirements and specifications

### Build Output
- **ELF**: `keil/Objects/empty_LP_MSPM0G3507_nortos_keil.axf`
- **HEX**: `keil/Objects/empty_LP_MSPM0G3507_nortos_keil.hex` (for flashing)
- **Rebuild**: Always clean before full rebuild if SysConfig changed
